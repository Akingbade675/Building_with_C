# ATM SIMULATION

### Pseudocode

```
1- start
2- Input your card
3- Type you pin
4- Display actions to perform e.g check balance, withdraw, transfers e.t.c
5- Type what action number to perform
6- perform action
7-Ask if user wants to another transaction
8- if number 6 is yes, goto number 3, else continue
9- stop
```
